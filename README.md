# 🐟 봉빵랜드 프렌즈 메이커! (BungOPang_Game)
> **모든 붕어빵의 천국 '팥소 언덕'에서 펼쳐지는 횡스크롤 러닝 액션**

**봉빵랜드 프렌즈 메이커!** 는 2024년 컴투스플랫폼 주관 '붕어빵 유니버스' 공모전을 위해 제작된 유니티 기반 횡스크롤 러닝 게임입니다. 짧은 개발 기간 동안 시스템의 안정성과 유저 경험의 디테일을 높이는 '내실 고도화'에 집중하여 완성되었습니다.

## 📖 게임 스토리 (Story)
모든 붕어빵의 신 **'붕신'** 이 만든 천국 팥소 언덕에 히어로를 꿈꾸는 붕어빵이 태어납니다. 붕신은 이 붕어빵에게 자신의 대리인이 되어 친구들을 창조해달라는 임무를 맡기고, 붕빵이는 친구들을 만들 재료(팥)를 모으기 위해 신비로운 붕어빵틀과 함께 모험을 떠납니다. 

## 🚀 주요 특징 (Key Features)

- **묵직한 비행 조작감**: 기존 러닝 게임의 가벼운 느낌과 차별화되는 **무게감 있는 비행 조작**을 구현하여 정교한 컨트롤의 재미를 줍니다.
- **역동적인 배경 연출**: 플레이 거리에 따라 마치 **계절이 변하는 듯한 실시간 배경 전환** 시스템을 적용했습니다.
- **전략적 아이템 시스템**:
  - **쉴드(Shield)**: 장애물 충돌 시 피격을 1회 방어합니다.
  - **플렉스(FLEX)**: 일정 시간 동안 일반 팥을 가치가 높은 **무지개 팥**으로 변화시켜 고득점을 노릴 수 있습니다.
  - **수집 및 커스터마이징**: **30여 종의 다양한 붕어빵 아이콘**을 제공하며, 획득한 재화로 캐릭터 외형을 변경할 수 있는 도감 시스템을 갖추고 있습니다.

## 🛠 개발 환경 및 기술 스택 (Tech Stack)

- **Engine**: Unity 2023.2.11f1
- **Language**: C#
- **Architecture**: Singleton 및 Manager 패턴을 활용한 안정적인 데이터 관리
- **Input**: 스페이스바를 활용한 직관적인 비행 높이 조절 시스템

## 🕹 조작 방법 (Controls)

- **스페이스바 (Space)**: 누르면 상승, 떼면 하강하여 장애물을 피하고 재화(팥)를 획득합니다.
- **아이템 선택**: 게임 시작 전 로비에서 획득한 아이템을 장착하여 전략적으로 시작할 수 있습니다.
